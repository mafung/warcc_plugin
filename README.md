# warcc - Vite + React + Tailwind CSS + TypeScript

This is a Vite-powered React application with TypeScript and Tailwind CSS.

## Project Structure

- `src/` - Source code directory
- `public/` - Static assets
- `tailwind.config.ts` - Tailwind CSS configuration
- `postcss.config.js` - PostCSS configuration
- `vite.config.ts` - Vite configuration

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint

## Features

- React with TypeScript
- Tailwind CSS for styling
- Vite as the build tool
- ESLint for code quality

## Getting Started

1. Install dependencies: `npm install`
2. Start development server: `npm run dev`

## Usage

To use Tailwind CSS in your components:

```jsx
<div className="bg-blue-500 text-white p-4">
  PrayItems
</div>
```

The project is now configured to use Tailwind CSS classes in your React components.
